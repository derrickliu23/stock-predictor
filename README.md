# Stock Price Predictor

Predicts next-day stock prices using an LSTM neural network trained on historical price and volume data. Includes a trading simulator that backtests buy/sell signals against a buy-and-hold benchmark, and a Streamlit dashboard to visualize everything.

## How it works

1. **Fetch** — pulls up to 10 years of historical OHLCV data from Yahoo Finance
2. **Engineer features** — computes moving averages, RSI, momentum, and volatility indicators
3. **Train** — trains a two-layer LSTM on 30-day sliding windows, predicting next-day % price change
4. **Predict** — loads the trained model and outputs a next-day price prediction with accuracy metrics
5. **Backtest** — simulates trades over the test set using predicted signals, tracks portfolio vs buy-and-hold
6. **Visualize** — Streamlit dashboard with prediction metrics, predicted vs actual chart, and backtest results

**2. Create a virtual environment**

## Usage

**Train a model for any ticker:**
```bash
python train.py AAPL
python train.py MSFT
python train.py TSLA
```

**Get a next-day prediction:**
```bash
python train.py AAPL predict
```

Output:
```
AAPL — 2026-06-05
Last close:      $307.34
Predicted next:  $307.14
Expected change: -0.20 (-0.06%)
```

**Evaluate model accuracy on the test set:**
```bash
python train.py AAPL evaluate
```

Output:
```
Test set evaluation:
MAE:  $2.73   (avg dollar error)
RMSE: $3.93   (penalizes big misses)
MAPE: 1.17%   (avg % error)
```

**Launch the dashboard:**
```bash
streamlit run app.py
```

Opens at `http://localhost:8501`.

## Dashboard

The dashboard has two tabs:

**Prediction tab**
- Next-day price prediction with expected % change
- MAE and MAPE accuracy metrics
- Predicted vs actual price chart over the test set
- Raw feature data table

**Backtest tab**
- Simulates trading over the test set using model signals
- Compares portfolio value vs buy-and-hold benchmark
- Buy/sell signal markers overlaid on the price chart
- Full trade log with dates, prices, and share counts
- Adjustable starting cash via the sidebar

## Features engineered

| Feature | Description |
|---|---|
| `ma_7 / ma_21 / ma_50` | Moving averages over 1 week, 1 month, 2 months |
| `roc_7 / roc_21` | Rate of change (momentum) over 7 and 21 days |
| `volatility_7 / volatility_21` | Rolling standard deviation of price |
| `volume_ratio` | Today's volume vs 7-day average — spike detector |
| `rsi` | Relative Strength Index (14-day) — overbought/oversold signal |
| `price_vs_ma21` | How far price sits above or below the 21-day trend |

## Model architecture

- Two stacked LSTM layers (32 → 16 units) with dropout
- Trained on 30-day sliding windows
- Predicts next-day **% price change** (not raw price) to avoid trend bias
- Custom directional loss (MSE + directional penalty) to prevent flat predictions
- Early stopping with patience of 20 epochs
- Trained with Adam optimizer

## Trading simulator

Buy/sell signals are generated by comparing the predicted % change against thresholds:

- Predicted change > +0.05% → **BUY** (all available cash)
- Predicted change < -0.30% → **SELL** (all held shares)
- Otherwise → **HOLD**

The backtester replays these signals over the test set (last 20% of historical data) and tracks portfolio value day by day, comparing against a simple buy-and-hold strategy.

## Results (AAPL, 10 years of data)

| Metric | Value |
|---|---|
| MAE | $2.73 |
| RMSE | $3.93 |
| MAPE | 1.17% |
| Model return (backtest) | +7.9% |
| Buy & hold return | +44.69% |

The model generates real returns but underperforms buy-and-hold — an honest and common result that reflects how difficult it is to beat passive investing with technical indicators alone.
